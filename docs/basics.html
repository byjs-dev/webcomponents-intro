What are Web Components?

Web components consist of three different technologies that are often used together:

  1. Custom Elements - Custom elements allow web developers to define new HTML tags, extend existing ones, and create reusable UI components.
  2. Shadow DOM - Shadow DOM provides a way for an element to own, render, and style a chunk of DOM that's separate from the rest of the page.
  3. HTML template - allows you to declare fragments of the DOM which are parsed, insert at page load, and can be activated later at runtime.
